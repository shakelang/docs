ShasamblyInterpretingBase

abstract class ShasamblyInterpretingBase(memorySize: Int, bytes: ByteArray, position: Int = 0)(source)

The base of the shasambly interpreter. It contains the core functions of the interpreter that store data and are used to manipulate the interprets data. It does not contain any functions that are used to interpret the data or to execute opcodes.
